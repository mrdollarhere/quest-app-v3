# Customizing Your Platform

You can make the platform your own without touching any code.

## Branding
Go to **Admin -> Settings**.
- **Platform Name**: This changes the text in the top left and on certificates.
- **Logo URL**: Paste a link to an image (ending in .png or .jpg). This will replace the default icon.
- **Theme Color**: Enter a Hex code (like `#FF0000` for red) to change the primary button colors across the whole site.

## Announcement Banner
Want to tell students about a deadline?
1.  Go to **Settings**.
2.  Type your message in **"Announcement Banner"**.
3.  It will appear at the very top of the homepage in bright blue.

## Access Keys
If you want to keep your tests private:
1.  Enable **"Protection Enabled"** in Settings.
2.  The site will now ask for a password every day.
3.  You can find the "Daily Key" on your Admin Dashboard. It changes every midnight!
