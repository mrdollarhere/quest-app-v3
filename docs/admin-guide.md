# Admin Console Guide

The Admin Console is your private control center. Only users with the `admin` role can see this.

## 1. Creating a Test
1.  Go to **Test Library** in the sidebar.
2.  Click **"New Test"**.
3.  Fill in the Title, Category (e.g., Math), and Difficulty.
4.  **Certification**: If you want students to get a PDF when they pass, toggle this ON and set a passing score (e.g., 70%).

## 2. Adding Questions
Once a test is created, click the **Questions** button next to it.
- **Add One**: Use the visual editor to build a question.
- **JSON Editor**: If you have a list of questions from an AI (like ChatGPT), you can paste the whole list here at once.

## 3. Managing Students
Go to the **Students** tab.
- **Add One**: Create a single account for a specific student.
- **Add Many (Batch)**: Create 50 accounts at once (e.g., student01, student02...) with one click. Useful for classrooms!

## 4. Viewing Results
Go to the **Results** tab.
- You can see a list of every test taken.
- Click on a student's name to see their personal history and charts.
- You can delete accidental test submissions here.

## 5. System Settings
Go to the **Settings** tab.
- **Branding**: Change your site name and logo.
- **Security**: Change the "Daily Key" salt.
- **Access Control**: Decide if guests can take tests or if accounts are required.
- **Maintenance**: Turn off the site temporarily while you update tests.
