## What is DNTRNG?
DNTRNG (pronounced "Dan-Truong") is a professional assessment tool that uses **Google Sheets** as its database. This means you own your data, and it costs $0 to host. It's built for schools and businesses that need high-quality interactive tests without complex technical overhead.
